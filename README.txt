# urospenezic.BookingWebApp
Razor Page Web Application for booking.
Requires .NET core 2.2 to run. Link to SDK: https://dotnet.microsoft.com/download/visual-studio-sdks
Right click on solution icon and open with Visual Studio Version Selector to ensure that it's being built in .net core 2.2
You will be given an admin account on startup and the database will be created under User folder of your system.
ADMIN ACCOUNT EMAIL: admin@gmail.com
ADMIN ACCOUNT PASSWORD: adminpassword
Feel free to create some restaurants, register users and reserve your seat. Enjoy.
